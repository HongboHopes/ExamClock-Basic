# 监考时钟基础版 ExamClock Basic

一个单文件的 HTML 考试监考计时工具，双击浏览器打开即用，无需安装，无需联网。

## 功能

- 实时北京时间显示（Asia/Shanghai 时区）
- 考试信息录入（科目、时间、试卷页数/张数、答题卡张数、监考人）
- 考试倒计时提醒（距结束15分钟变红+音频）
- 考试结束自动提示
- 全屏模式（教室投影大屏适配）
- 音量调节
- 自适应分辨率

## 使用方法

1. 双击 `index.html` 用浏览器打开
2. 填写右侧考试信息
3. 点击"开始考试"
4. 考试过程全自动执行

## 技术栈

- 纯 HTML/CSS/JavaScript
- 无外部依赖
- 图片和音频 Base64 内嵌
